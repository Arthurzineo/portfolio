# Arthur Gonçalves — Portfolio

Portfólio profissional bilíngue voltado a Security Engineering e AI Security.

## Tecnologias

- TypeScript
- React
- Next.js App Router
- Vinext e Vite
- Cloudflare Workers

## Desenvolvimento local

Requisitos: Node.js 22.13 ou superior.

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
```

O projeto gera o artefato de execução para Cloudflare Workers dentro de `dist/`.
